Question: Trace billing document 91150187
```cypher
MATCH (b:BillingDocument)
WHERE b.id CONTAINS '91150187'
OPTIONAL MATCH (s:SalesOrder)-[:HAS_BILLING]->(b)
OPTIONAL MATCH (c:Customer)-[:PLACED]->(s)
OPTIONAL MATCH (s)-[:HAS_DELIVERY]->(d:Delivery)
OPTIONAL MATCH (b)-[:POSTED_TO]->(j:JournalEntry)
OPTIONAL MATCH (p:Payment)-[:CLEARS]->(j)
RETURN
  c.fullName AS customer,
  s.id AS salesOrder,
  s.totalAmount AS orderAmount,
  b.id AS billingDocument,
  b.totalAmount AS billingAmount,
  b.isCancelled AS isCancelled,
  d.id AS delivery,
  d.goodsMovementStatus AS deliveryStatus,
  j.id AS journalEntry,
  j.postingDate AS postingDate,
  p.id AS payment,
  p.amount AS paymentAmount
LIMIT 10
```

Question: Which customer placed the most orders?
```cypher
MATCH (c:Customer)-[:PLACED]->(s:SalesOrder)
RETURN
  c.fullName AS customer,
  c.id AS customerId,
  COUNT(s) AS orderCount,
  SUM(toFloat(s.totalAmount)) AS totalValue
ORDER BY orderCount DESC
LIMIT 5
```

Question: Find sales orders with no billing documents

Question: Find incomplete order flows
```cypher
MATCH (s:SalesOrder)
OPTIONAL MATCH (s)-[:HAS_DELIVERY]->(d:Delivery)
OPTIONAL MATCH (s)-[:HAS_BILLING]->(b:BillingDocument)
OPTIONAL MATCH (b)-[:POSTED_TO]->(j:JournalEntry)
OPTIONAL MATCH (c:Customer)-[:PLACED]->(s)
WITH s, c,
     COUNT(d) AS hasDelivery,
     COUNT(b) AS hasBilling,
     COUNT(j) AS hasJournal
WHERE hasDelivery = 0
   OR hasBilling = 0
   OR hasJournal = 0
RETURN
  s.id AS salesOrder,
  c.fullName AS customer,
  s.totalAmount AS amount,
  CASE WHEN hasDelivery = 0 THEN 'Missing Delivery' ELSE 'Has Delivery' END AS deliveryStatus,
  CASE WHEN hasBilling  = 0 THEN 'Missing Billing'  ELSE 'Has Billing'  END AS billingStatus,
  CASE WHEN hasJournal  = 0 THEN 'Missing Journal'  ELSE 'Has Journal'  END AS journalStatus
ORDER BY s.totalAmount DESC
LIMIT 20
```

Question: How many of each entity type exist?
```cypher
MATCH (c:Customer)       WITH COUNT(c) AS customers
MATCH (s:SalesOrder)     WITH customers, COUNT(s) AS orders
MATCH (d:Delivery)       WITH customers, orders, COUNT(d) AS deliveries
MATCH (b:BillingDocument) WITH customers, orders, deliveries, COUNT(b) AS billingDocs
MATCH (p:Payment)        WITH customers, orders, deliveries, billingDocs, COUNT(p) AS payments
MATCH (j:JournalEntry)   WITH customers, orders, deliveries, billingDocs, payments, COUNT(j) AS journals
RETURN customers, orders, deliveries, billingDocs, payments, journals
```

Question: Show cancelled billing documents

```cypher
MATCH (b:BillingDocument)

OPTIONAL MATCH (s:SalesOrder)-[:HAS_BILLING]->(b)
OPTIONAL MATCH (c:Customer)-[:PLACED]->(s)
OPTIONAL MATCH (b)-[:CANCELLED_BY]->(cb:BillingDocument)
RETURN
  b.id AS originalBilling,
  b.totalAmount AS amount,
  c.fullName AS customer,
  s.id AS salesOrder,
  cb.id AS cancelledBy
ORDER BY b.creationDate DESC
LIMIT 20
```

Question: What is total revenue by customer?
```cypher
MATCH (c:Customer)-[:PLACED]->(s:SalesOrder)-[:HAS_BILLING]->(b:BillingDocument)
WHERE b.isCancelled = false OR b.isCancelled = 'false'
RETURN
  c.fullName AS customer,
  COUNT(DISTINCT s) AS orders,
  COUNT(DISTINCT b) AS billingDocs,
  SUM(toFloat(b.totalAmount)) AS totalRevenue
ORDER BY totalRevenue DESC
LIMIT 20
```
